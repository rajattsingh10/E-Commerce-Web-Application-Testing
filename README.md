# Project 1 - E-Commerce Web Application Testing

## Application Under Test
https://www.saucedemo.com/

## Objective
To validate core e-commerce functionalities including:
- Login
- Product listing
- Add to cart
- Cart persistence
- Checkout
- Form validation
- UI/UX behavior

- Bug Report (Excel)
- Test Execution Report

  - Actual: Error message covers the login button and breaks layout
